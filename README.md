📄 README - Manual de Instalação do Windows 11 em Máquina Virtual\
📚 Enquadramento\
Este projeto foi desenvolvido no âmbito da unidade curricular UC00629 – Instalar e Parametrizar Sistemas Operativos Plataforma Proprietária, integrada no curso de Especialização Tecnológica (CET) em Técnico/a Especialista em Gestão de Redes e Sistemas Computacionais.\
O objetivo consiste em demonstrar, de forma prática e documentada, o processo completo de instalação e configuração do sistema operativo Windows 11 numa máquina virtual utilizando o Oracle VirtualBox.
________________________________________
🎥 Descrição dos Vídeos\
🔹 Fase 1 - Configuração da Máquina Virtual\
Vídeo: Fase 1.mp4\
Demonstra a criação inicial da máquina virtual no VirtualBox, incluindo a definição do nome, localização e associação da imagem ISO do Windows 11. São também configurados os parâmetros-base do sistema operativo.\
Apresenta a configuração do hardware da máquina virtual, incluindo a alocação de memória RAM, número de processadores, ativação do EFI e definição do disco virtual. Inclui ainda a verificação das opções de aceleração e armazenamento.
________________________________________
🔹 Fase 2 - Instalação do Sistema Operativo\
Vídeo: Fase 2.mp4\
Mostra o processo de instalação do Windows 11, desde a seleção do idioma e teclado até à criação e gestão de partições no disco. Inclui o progresso automático da instalação do sistema operativo.
________________________________________
🔹 Fase 3 - Configuração Pós-Instalação\
Vídeo: Fase 3.mp4\
Demonstra a criação da conta de utilizador, incluindo a definição do nome, palavra-passe e perguntas de segurança, essenciais para a gestão do sistema.\
Apresenta a configuração das opções de privacidade do Windows 11, como localização, dados de diagnóstico e preferências de personalização.
________________________________________
🔹 Fase 4 - Finalização e Otimização\
Vídeo: Fase 4.mp4\
Mostra a instalação do VirtualBox Guest Additions e verificar as definições do sistema as especificações do dispositivo (como o processador e as especificações do Windows.
________________________________________
🎯 Objetivo do Projeto\
•	Demonstrar competências na instalação de sistemas operativos proprietários
•	Configurar corretamente uma máquina virtual
•	Aplicar boas práticas de particionamento e configuração
•	Validar o funcionamento do sistema após instalação
•	Melhorar a integração entre sistema anfitrião e máquina virtual
________________________________________
🛠️ Tecnologias Utilizadas\
•	Oracle VirtualBox\
•	Windows 11\
•	Sistema anfitrião Windows\
•	HTML (para apresentação web do relatório)
________________________________________
🚀 Nota Final\
Este projeto evidencia a importância da virtualização como ferramenta essencial na administração de sistemas, permitindo testar, configurar e validar ambientes de forma segura e controlada.
